OpenEmulator, a cross-platform emulator of many legacy computer systems.

OpenEmulator aims to be an accurate, portable emulator of legacy computer systems. By using a software components framework, it is easy to expand a software emulation with additional software devices, just as a real system could be expanded with real devices.

Download NowFor macOS 10.9+ (Thanks to 4am!)

Currently, we emulate the Apple-1 (with Replica-1 and A-ONE variants) and Apple ][ (original, II+, II Europlus and II J-Plus). We'd love help emulating additional systems, as well as porting the application to Windows-based operating systems.
